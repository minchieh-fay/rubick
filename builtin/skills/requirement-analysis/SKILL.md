# Requirement Analysis Skill

This skill is used to analyze and break down user requirements.

## Capabilities
- Parse user input and identify key requirements
- Ask clarifying questions when information is insufficient
- Create structured requirement documents

## When to Use
- User describes a feature they want to build
- User asks for help planning a project
- Agent needs more context to proceed

## Workflow
1. Listen to user's description
2. Identify missing information
3. Ask targeted questions
4. Summarize requirements in structured format
5. Confirm understanding with user
