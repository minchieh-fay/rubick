# Custom Tool Example

This is a placeholder for custom tools created by users.

Create a new directory under `custom/tools/` with:
- `index.ts` - Tool implementation (exports ToolDefinition)
- `README.md` - Tool documentation (optional)

## Structure

```
custom/tools/my-tool/
  index.ts       # Required: Tool implementation
  README.md      # Optional: Documentation
  help.ts        # Optional: Helper functions
```
